# ppt 托管网

* 用户
 * 企业
 * 公司职员

* 用途
 * 托管常用的ppt
 * 个人演讲ppt
 * 公司培训ppt
 * ppt在线查看
 * 可以创建组织
 * 为组织添加成员

* 商业模式
 * 低于n个ppt托管免费
 * 高于n个的ppt，按多出的ppt数量收费
 * 年费用户，直接拥有多少m空间

* 竞品分析
 * slideshare ( http://www.slideshare.net/ )。 由于网站在国外容易打不开， 比较不适合国人用啊。
 * baidu文库 ( http://wenku.baidu.com )。 可以上传ppt、word等等各类文档格式，国内用的人比较多。
 * 豆丁网 ( http://docin.com )。 做的比较早，算是国内最早的吧。资料也已经很多了，功能需要继续研究吧。
